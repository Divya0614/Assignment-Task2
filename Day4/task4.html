Description list

<HTML>
The root element of an HTML document.It wraps all other content including head and body tags
<head>
Contains meta information about the document like title , links to stylesheets and scripts, it is not displayed directly on the page.
<title>
Defines the title of the webpage, which appears in the browser tab and it is important for SEO.
<body>
Contains all visible content of the webpage, including text, images, link, tables, list and forms
<p>
Reprsenst a paragraph of text, it automatically adds spacing before and after the text fro better readability
<a>
Used to create a hyperlinks that link to other web pages, email address or files, the destination is specified using the href attribute
<img>
Displays images on th ewebpage , uses attibutes like srcfor the image url and alt for alternative text
